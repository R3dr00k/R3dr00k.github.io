Fonctionnement

Le Tri par insertion parcourt chaque élément du tableau puis les inserent à la bonne place parmi les élément déja triés .
Pour trouver la bonne place parmi les éléments déja triés il faut le comparer à chaque éléments de la partie triée . SI l'élément est plus grand que tous les éléments trié alors il suffit de continuer à parcourir le tableau car cet élément est déja trié . SINON il faut decaler les élements trié pour laisser un espace vide où inserer l'élément .

Exemple

Prenons par exemple un tableau : [6, 8, 5, 3, 9]:
1ère itération i=1:
[6, 8, 5, 3, 9] => [6, 8, 5, 3, 9]
i=2:
[6, 8, 5, 3, 9] => [5, 6, 8, 3, 9]
i=3:
[5, 6, 8, 3, 9] => [3, 5, 6, 8, 9]
i=4 and last:
[3, 5, 6, 8, 9] => [3, 5, 6, 8, 9]

Pseudo Code


tri_insertion(tableau T)
     pour i de 1 à taille(T) - 1

        # mémoriser T[i] dans x
        x ← T[i]                            

        # décaler les éléments T[0]..T[i-1] qui sont plus grands que x, en partant de T[i-1]
        j ← i                               
        tant que j > 0 et T[j - 1] > x
                 T[j] ← T[j - 1]
                 j ← j - 1

        # placer x dans le "trou" laissé par le décalage
        T[j] ← x

Implémentation en Python

En python on peut utiliser les méthodes de list insert(index, value) et pop(index) qui nous évite d'avoir à décaler les valeurs triées.


def tri_insertion(tab):
    for i in range(1,len(tab)):
        for j in range(0, i):
            if tab[i] <= tab[j]:
                tab.insert(j, tab.pop(i))
                break
    return tab

Cet Algorithme parcours chaque élément du tableau, puis cherche dans la partie triée le bon emplacement pour ces derniers.
