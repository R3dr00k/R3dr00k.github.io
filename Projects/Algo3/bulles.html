Fonctionnement

Le tri a bulle compare chaque élément au suivant puis les permutent lorsqu'ils sont mal triés, il répéte cette opération jusqu'a ce que le tableau soit totalement trié.

Exemple

Prenons un tableau : [5, 1, 4, 2, 8]

i=0 ( 5 1 4 2 8 ) → ( 1 5 4 2 8 ). Les nombres 5 et 1 sont comparés, et comme 5 > 1, l'algorithme les permute.
i=1 ( 1 5 4 2 8 ) → ( 1 4 5 2 8 ). Permutation, car 5 > 4.
i=2 ( 1 4 5 2 8 ) → ( 1 4 2 5 8 ). Permutation, car 5 > 2.
i=3 ( 1 4 2 5 8 ) → ( 1 4 2 5 8 ). Pas de permutation, car 5 < 8.
On récommence jusqu a ce que le tableau soit trié

Pseudo Code


tri_à_bulles_optimisé(Tableau T)
    pour i allant de (taille de T)-1 à 1
        tableau_trié := vrai
        pour j allant de 0 à i-1
            si T[j+1] < T[j]
                (T[j+1], T[j]) = (T[j], T[j+1])
                tableau_trié := faux
        si tableau_trié
            fin tri_à_bulles_optimisé

Implémentation en Python


def tri_bulles(tab):
    sorted = False
    while not sorted:
        sorted = True
        for i in range(len(tab)-1):
            if tab[i] > tab[i+1]:
                tab[i], tab[i+1] = tab[i+1], tab[i]
                sorted = False
    return tab
